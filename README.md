# React CRUD Application

## Prerequisites

- [MongoDB Community Server](https://www.mongodb.com/try/download/community)
- Node.js

---

## 1. Start MongoDB

```bash
mkdir C:\data\db
```

```bash
"C:\Program Files\MongoDB\Server\8.2\bin\mongod.exe" --dbpath="C:\data\db"
```

---

## 2. Start Backend

```bash
cd backend
```

```bash
node db.js
```

```bash
node app.js
```

---

## 3. Start Frontend

```bash
cd frontend
```

```bash
npm install
```

```bash
npm start
```

> `npm run dev` — ไม่จำเป็นต้องใช้
